¿Cómo funciona el proceso de voto?

Sigue estos pasos para ejercer tu voto de manera segura, anónima y auditable.

1. Generar Llave

El sistema generará un par de llaves criptográficas únicas para ti. Se descargará automáticamente un archivo (tu llave privada).
¡Guarda este archivo! Lo necesitarás para firmar tu voto.

2. Emitir Voto

Responde la encuesta y adjunta el archivo de llave privada que descargaste en el paso anterior. El sistema usará este archivo para firmar digitalmente tu elección, garantizando que eres tú sin revelar tu identidad.

3. Tablero Público

Una vez procesado, puedes ver las gráficas en tiempo real. Podrás observar cómo se distribuyen los votos de todos los participantes de manera transparente.

4. Verificación

Al finalizar, obtendrás un Comprobante Digital. Puedes ir a la sección "Verificar mi Voto" para confirmar que tu participación se encuentra registrada.

Herramienta: Validar Llave

¿Tienes dudas sobre tu archivo? Puedes usar esta herramienta antes o después de votar. Verifica si tu llave es auténtica, si corresponde a tu usuario y su estado actual.
